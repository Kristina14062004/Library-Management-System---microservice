# 📚 Library Management System (Microservices)

## 🚀 Overview
This project is a **Library Management System** built using a **microservices architecture**. It allows efficient management of books and related operations through scalable and modular services.

The system is developed using Spring Boot and follows RESTful principles for communication between services.

---

## 🧩 Architecture
The application is divided into multiple independent services, making it:
- Scalable 📈  
- Maintainable 🛠️  
- Easy to extend 🔄  

Service discovery is handled using Netflix Eureka.

---

## ⚙️ Tech Stack
- Backend: Spring Boot  
- Language: Java  
- Architecture: Microservices  
- Service Discovery: Netflix Eureka  
- API: REST APIs  

---

## ✨ Features
- 📖 Add new books  
- 🔍 View and search books  
- ✏️ Update book details  
- ❌ Delete books  
- 🔗 Microservices-based modular design  
- 📡 Service discovery using Eureka
